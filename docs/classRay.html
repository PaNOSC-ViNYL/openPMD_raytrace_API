Detailed Description

Generic ray, containing all the ray information being stored by the API into the openPMD file.

This is base class provides several methods to set and manipulate the ray information in the appropriate way Other specific ray classes should inherit from this in order to provide more convinient methods for the user. Derived classes can be created from this and used by the users in their programs
