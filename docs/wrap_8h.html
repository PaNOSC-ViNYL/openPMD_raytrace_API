typedef struct openPMD_io openPMD_io

openPMD_io struct, defined from the openPMD_io class

The openPMD C++ API is used by a class ("openPMD_io") dedicated to perform I/O of McStas simulation in openPMD format. This file provides a wrapper to the openPMD_io class in order to be used in McStas as an external library.
